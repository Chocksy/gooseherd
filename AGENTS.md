# Repository Guidelines

## Commit Messages

- Name commits to satisfy the conventional-commit rules enforced by `commitlint`.
- Start the commit subject after the conventional-commit prefix with a capital letter, e.g. `feat: Add model prices`.
- Reference: https://github.com/conventional-changelog/commitlint/#what-is-commitlint
- Local rules for this repository live in `commitlint.config.mjs`.
